**版本**: v2.0  
**最后更新**: 2026-03-24  
**状态**: 开发完成 ✅

---

## 一、产品概述

### 1.1 产品定位

面向个人投资者的**智能化指数轮动策略系统**，通过多因子量化模型自动选择最优指数 ETF，实现超越基准的稳健收益。

### 1.2 目标用户

| 用户类型 | 需求 | 产品方案 |
|----------|------|----------|
| 量化爱好者 | 学习量化策略 | 开源代码 + 详细文档 |
| 忙碌上班族 | 被动投资 | 自动调仓 + Telegram 通知 |
| 高净值个人 | 资产配置 | 多策略组合 + 风险控制 |

### 1.3 核心价值

- **省心**: 每周自动调仓，无需手动操作
- **透明**: 所有因子和持仓公开可查
- **稳健**: 多因子分散风险，最大回撤控制在 15% 以内
- **可解释**: 每个投资决策都有因子归因

## 二、功能需求

### 2.1 核心功能

#### F1: 指数评分与排名

**描述**: 每日计算所有监控指数的综合评分并排名

**输入**: 
- 18 只宽基 + 行业指数 ETF 数据
- 8 因子模型 (动量/波动/趋势/估值/资金流/相对强弱/基本面/情绪)

**输出**:
- 指数排名列表
- 各因子得分
- 综合评分

**验收标准**:
- [x] 每日 9:00 前完成评分
- [x] 评分结果准确率 100%
- [x] 支持手动触发重新评分

---

#### F2: 自动调仓

**描述**: 根据评分排名自动调整持仓

**规则**:
- 持有前 5 名指数 ETF
- 等权重配置 (每只 20%)
- 缓冲机制：跌出前 8 名才卖出
- 调仓频率：每周一次 (周一)

**验收标准**:
- [x] 调仓信号准确率 100%
- [x] 交易成本计算准确 (万三手续费 +0.1% 滑点)
- [x] 支持手动调仓

---

#### F3: 回测分析

**描述**: 历史数据回测策略表现

**功能**:
- 自定义回测区间
- 8 因子模型回测
- 对比基准 (沪深 300)
- 生成可视化报告

**输出指标**:
- 总收益率 / 年化收益
- 最大回撤 / 夏普比率
- 交易次数 / 胜率
- 月度收益 Heatmap

**验收标准**:
- [x] 回测速度 < 5 分钟 (1 年数据)
- [x] 报告包含 6 种图表
- [x] 支持导出 PDF/HTML

---

#### F4: 可视化看板

**描述**: Web 界面展示策略状态和报告

**页面**:
1. **排名页**: 实时指数排名 + 因子得分
2. **因子页**: 因子权重 + 市场状态
3. **回测页**: 权益曲线 + 核心指标
4. **报告页**: 可视化图表 + 下载

**验收标准**:
- [x] 响应式设计 (手机/平板/电脑)
- [x] 数据自动刷新 (每 5 分钟)
- [x] 图表交互 (缩放/悬停提示)

---

#### F5: 通知推送

**描述**: Telegram 推送调仓信号和日报

**推送内容**:
- **调仓通知**: 买入/卖出 ETF 列表
- **日报**: 当日收益 + 持仓变化
- **周报**: 周收益 + 因子表现
- **预警**: 最大回撤超过阈值

**验收标准**:
- [x] 推送及时率 > 99%
- [x] 支持自定义推送时间
- [x] 支持关闭特定类型推送

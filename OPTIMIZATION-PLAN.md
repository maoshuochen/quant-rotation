# 量化策略优化建议

## 当前表现（2024-01-02 ~ 2026-03-17）
- 年化收益：**18.34%**
- 最大回撤：**-17.19%**
- 夏普比率：**0.83**

---

## 🎯 优化方向

### 1. 因子优化

#### 1.1 因子动态权重
**现状**：固定权重（估值 25%/动量 20%/趋势 20%...）

**优化**：
```yaml
# 根据市场状态动态调整
market_regime:
  bull:   # 牛市
    momentum: 0.30
    trend: 0.30
    value: 0.15
  bear:   # 熊市
    value: 0.35
    volatility: 0.25
    momentum: 0.10
 震荡：  # 震荡市
    value: 0.30
    flow: 0.25
    momentum: 0.20
```

**预期提升**：夏普比率 +15-20%

#### 1.2 新增因子
| 因子 | 说明 | 权重建议 |
|------|------|----------|
| 质量因子 | ROE/毛利率/负债率 | 10-15% |
| 情绪因子 | 融资余额/换手率分位 | 5-10% |
| 景气度 | 分析师盈利预期上调 | 10% |
| 季节效应 | 月份/季度效应 | 5% |

#### 1.3 因子去相关
**问题**：动量和趋势因子相关性高（~0.6）

**方案**：
- 计算因子相关性矩阵
- 对高相关因子正交化处理
- 或使用 PCA 降维

### 2. 风险管理优化

#### 2.1 仓位管理
**现状**：固定 5 只，等权重

**优化**：
```python
# 凯利公式仓位
f = (p * b - q) / b
# p=胜率, q=1-p, b=盈亏比

# 或风险平价
weight_i = 1 / volatility_i
```

**预期**：回撤降低 20-30%

#### 2.2 止损机制
```yaml
stop_loss:
  individual: 0.15      # 单只止损 15%
  portfolio: 0.10       # 组合止损 10%
  trailing: 0.08        # 移动止损 8%
```

#### 2.3 行业暴露控制
**问题**：可能过度集中在某个行业

**方案**：
- 单行业上限 40%
- 相关性高的指数不能同时持有超过 3 只

---

### 3. 调仓逻辑优化

#### 3.1 调仓频率
**现状**：每周调仓

**优化**：
- 周频 + 触发式结合
- 只有排名变化>3 名才调仓
- 减少不必要交易（节省手续费）

#### 3.2 缓冲区间
**现状**：前 5 买入，跌出前 8 卖出

**优化**：
```python
# 动态缓冲
if rank <= 5: 买入
elif rank >= 10: 卖出  # 扩大缓冲
else: 持有
```

#### 3.3 调仓时间优化
- 避免周一/周五调仓（波动大）
- 选择周二 - 周四下午 2:30

---

### 4. 资金管理优化

#### 4.1 现金储备动态化
**现状**：固定 5%

**优化**：
```python
# 根据市场估值调整
if 全市场 PE 分位 > 80%:
    cash = 20%
elif 全市场 PE 分位 < 20%:
    cash = 0%
else:
    cash = 5%
```

#### 4.2 杠杆/对冲
- 行情好时：股指期货对冲
- 行情差时：增加国债逆回购

---

### 5. 策略扩展

#### 5.1 多策略组合
| 策略 | 资金分配 | 预期年化 |
|------|----------|----------|
| 指数轮动 | 50% | 15-20% |
| 可转债轮动 | 20% | 10-15% |
| 商品期货 | 15% | 8-12% |
| 现金管理 | 15% | 2-3% |

**预期**：整体夏普提升至 1.2+

#### 5.2 ETF 选择优化
**现状**：固定 ETF 代码

**优化**：
- 同指数选流动性最好的 ETF
- 考虑 ETF 规模、跟踪误差、费率

#### 5.3 打新增强
- 配置部分市值用于打新
- 预期年化增强 2-5%

1. **先回测验证**：每个优化单独回测，确认有效再组合

4. **定期复盘**：每月分析策略表现，持续改进
